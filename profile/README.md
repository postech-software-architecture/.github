# FIAP Pós Tech — Software Architecture (Grupo de Estudos)

Repositórios e workflow do nosso grupo (5 pessoas) para estudar e entregar os Tech Challenges por fase.

## Como trabalhamos
- **Fonte única de verdade:** GitHub Project (Kanban)
- **Trabalho executável:** Issues (com labels por fase/área/tipo)
- **Entrega:** Pull Requests + Review
- **Decisões:** ADRs (linkados em issue/PR quando necessário)

## Links rápidos
- Project (Kanban): [Board](https://github.com/orgs/postech-software-architecture/projects/1)

## Ritmo
- 1 sessão/semana: revisão + exercícios
- 1 sessão/semana: Tech Challenge (refino + code review + demo)
- Toda segunda: check-in de objetivo/bloqueios

## Convenções
### Labels
- `fase-01..fase-05`
- `tipo:duvida | tipo:resumo | tipo:exercicio | tipo:tech-challenge | tipo:task`
- `area:arquitetura | area:backend | area:devops | area:seguranca | area:dados`
- `prio:P0 | prio:P1 | prio:P2`

### ADR
- Local: `postech-sa-hub/adr/ADR-XXX-<titulo>.md`
- 1 decisão relevante = 1 ADR

## Como entrar
Peça convite para a organização informando seu **GitHub username** ao mantenedor.
